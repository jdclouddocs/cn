# 应用场景

**以下说明京东云DDoS基础防护的应用场景：**

京东云DDoS基础防护仅适用于京东云内的公网IP用户，并且攻击流量最高不超过2G的DDoS攻击防护场景。

如果您需要更大的攻击防护，或者有京东云外的DDoS防护需求，请购买"IP高防"。详情参考：[IP高防](http://www.jdcloud.com/products/anti-ddos-pro)
